# 黙走会 / minamo.run

**各自の場所で、黙って走る。主催者はいない。**

走った日に、走った証を一枚のページに静かに残す。それだけの場所です。
毎日でも、週に一度でも。歩いても、休んでも、やめてもいい。

- **01 わたしの積み重ね** — 累計だけが自分の端末に残る(非公開・localStorage のみ。サーバーには何も送らない)
- **02 今日の静かなつながり** — 「今日、共に走った人 n」。誰のものでもない数字。明日にはゼロから
- **03 会釈** — 名前のない光の点に、一日に一度だけそっと触れられる。誰か分からない、返せない、1ビットの会釈

連絡手段なし・ランキングなし・ログインなし・広告なし・課金なし。
連続日数(ストリーク)はありません。**休んでも、減りません。**

本番ページ: https://minamo-eyu.pages.dev (minamo.run へ移行予定)

## このページごと複製して、あなたの街の黙走会にしてよい

憲章7条 — この型は誰のものでもない。誰でも、どこでも、許可なく開催してよい。
連絡も報告も不要です(そもそも、この会には連絡手段がありません)。

1. **`public/` フォルダごとコピーして、`index.html` を開く。** これだけで層1(わたしの積み重ね)が動きます。
   ファイル取り込み(GPX / TCX / FIT)もブラウザ内で完結します
2. そのままでも、あなたの複製から層2・3が元の水面に混ざることはありません(オリジン境界で静かに畳まれ、層1だけが動きます)。
   行儀よくするなら `index.html` 末尾のスクリプトの `API_BASE` を空文字 `''` に
3. あなたの街にも「今日、共に走った人」と会釈を灯すなら、`worker/` を自分の Cloudflare アカウントに
   デプロイして `API_BASE` を差し替えます(無料枠で足ります) → [docs/05_デプロイ手順.md](docs/05_デプロイ手順.md)
4. あなたの街のタイムゾーンに合わせるなら、`public/js/minamo-core.js` と `worker/src/index.js` の
   `TIME_ZONE` を同じ値に(例: `'America/New_York'`)。両者がズレると層1と層2/3で日の変わる瞬間が
   分かれます(データ喪失はなく、TTLで消える範囲の静かなズレに留まります)
