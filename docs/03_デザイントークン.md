### 1b. 未明の紙 — ダークトークン（2026-07-09 追加）

夜明け前に走る人(小川ペルソナ)の暗い部屋を眩しくしないための `prefers-color-scheme: dark` 上書き。
**朱(--accent)と灯(--glow / --glow-ink)は変えない** — 夜でも印は朱、灯は灯。
切替UIは作らない(OSの設定に従うだけ。設定という概念を持ち込まない)。

```css
@media (prefers-color-scheme: dark) {
  :root {
    --bg:            #211e23;                 /* 未明の墨紙(わずかに紫がかった墨) */
    --paper:         #29252b;                 /* カード地 */
    --ink:           #e6ddcd;                 /* 生成りの文字 */
    --sub:           #998c7c;
    --line:          rgba(230,221,205,.14);
    --accent-ink:    #d98a55;                 /* 暗地では少し明るい朱に */
    --accent-soft:   rgba(221,132,73,.45);
    --btn-shadow:       0 7px 16px rgba(12,6,3,.5);     /* 真っ黒の大きな影は使わない(足元が汚れる) */
    --btn-shadow-press: 0 3px 6px rgba(12,6,3,.55);
    --card-shadow:      0 2px 10px rgba(0,0,0,.3);
    --night:         #18161d;                 /* 会釈カードはさらに深い夜 */
    --field:         rgba(255,255,255,.06);   /* 入力欄の地 */
  }
  /* 明転は残しつつ暗所での眩しさを抑える(α .9 → .55) */
}
```

`--field`(入力欄の地)はライト側にも追加した: `rgba(255,255,255,.55)`。

## 4. アニメーション（この7つだけ。後3つは押印の所作・一度きり）

```css
@keyframes mkFlash   { 0%{opacity:0} 14%{opacity:.95} 45%{opacity:.7} 100%{opacity:0} }   /* 明転 1.25s ease-out forwards */
@keyframes mkPulse   { 0%,100%{transform:scale(1);opacity:.85} 50%{transform:scale(1.3);opacity:1} }  /* 会釈の灯り 2.6s〜3s */
@keyframes mkAppear  { from{opacity:0;transform:translateY(6px)} to{opacity:1;transform:translateY(0)} } /* 言葉・エディタの出現 .6〜.7s */
@keyframes mkTwinkle { 0%,100%{filter:brightness(.85)} 50%{filter:brightness(1.25)} }      /* 点の明滅 2.4〜5.4s（点ごとに乱数） */
@keyframes mkPressLift { /* 判が持ち上がる .5s（押印の所作・press-seq） */ }
@keyframes mkPon       { /* 今日の印がカードにぽんと着地 .42s cubic-bezier(.2,1.35,.45,1)（pressed-now） */ }
@keyframes mkDip       { /* カードが受け止めて少し沈む .34s */ }
```

- 明転はJS側で1350ms後に要素を除去（`mkFlash 1.25s ease-out forwards`）
- ボタン押下: `:active { transform: translateY(4px) scale(.97,.95); }` + 影が --btn-shadow-press に締まる
- 押印の所作は押したセッションだけ動く（リロードでは再生しない。press-seq → 430ms → pressed-now）
- **常時動く装飾は点の明滅のみ**。ループアニメーションを追加しない
- `@media (prefers-reduced-motion: reduce)` で全アニメーションを無効化する

## 5. 点（光）の疑似乱数 — 原本から移植する関数

同じ日は誰が見ても同じ星空になるよう、日付文字列をシードにする決定的乱数を使う
（FNV-1a ハッシュ → mulberry32。`design-reference/MokusoApp.dc.html` の `rng(seedStr)` をそのまま移植。
シードは `todayKey()` の日付文字列）。

## 6. スタンプの決定的乱数（2026-07-10 改稿 — 旧・落款の漢数字変換を置き換え）

- n日目のスタンプはシード `minamo-stamp/n` の rng で描画（絵柄4種の選択 → 角度±6.5° → 位置±2px → 濃さ）
- シードが日付でなく**累計番号**なので、一度押された印は翌日以降も同じ姿（粒と同じ「動かない過去」）
- カードの重なりはシード `minamo-pile` 固定（佇まいが毎日変わらない）
- 絵柄: 走の印（丸枠+走）/ 花丸（波の外周+渦）/ 朝日（半円+地平線+光条5本）/ 水面（一粒の月+波紋3本）。
  いずれも stroke ベースの澄んだ線。かすれ・グランジ加工はしない
