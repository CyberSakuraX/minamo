## 1. 層1 と JST日付

- [ ] 押印 → DevTools Application タブで `minamo.total / last / note / sumKm` を目視確認
- [ ] **日付ロールオーバー試験**: コンソールで `localStorage.setItem('minamo.last','2026-07-11')` にして再読込
      → ボタンが再び押せる / total は保持される / 「途切れ」的な表示・文言が一切出ない
- [ ] **JST境界の単体確認**（コンソールで実行）:
      ```js
      new Intl.DateTimeFormat('en-CA',{timeZone:'Asia/Tokyo'}).format(new Date('2026-07-12T14:59:00Z')) // → "2026-07-12"
      new Intl.DateTimeFormat('en-CA',{timeZone:'Asia/Tokyo'}).format(new Date('2026-07-12T15:00:00Z')) // → "2026-07-13"
      ```
      （UTC 15:00 = JST 0:00 で日付が変わる）。クライアントと Worker が**同一式**であることをコードレビューで確認
- [ ] 添え書きの書き直し: 5km で保存 → 3km に書き直し → sumKm が +3 になっている（+8 になっていない）
- [ ] 落款の漢数字: 2026-07-12 に「二〇二六」「七月十二日」と出る（10日・20日・21日等の桁も机上確認）

## 2. 層2（今日の人数）

- [ ] `curl -X POST <API>/api/stamp` を2回 → `curl <API>/api/today` で count=2・dots に2件
- [ ] 5分キャッシュの遅延を仕様として確認（即時反映しないことは正常。「数字は目安」）
- [ ] `wrangler kv key list --binding MINAMO_KV` で**全キーに expiration が付与**されている
- [ ] 翌日（またはTTL経過後）に同コマンドでキーが消滅している → **リセット処理ゼロの実証**

## 3. 層3（会釈の送受信）

- [ ] 窓A（通常）と窓B（シークレット）で各々押印 → Aの点フィールドに他者の点が現れる
      → Aが一点をタップ → Bを再読込 → 「今日のあなたの走りに、誰かが会釈しました。」が灯る
- [ ] **受信側に数がどこにも出ない**（バナーは文言のみ・点の数と受信は無関係）
- [ ] Aが同日中に2点目をタップできない（キャプションが「そっと、会釈を残しました。〜」に変わっている）
- [ ] curl で同一 `to` に POST /api/bow を連打 → Bの表示は変わらない（1ビットの実証）
- [ ] 存在しない `to` への bow が 404 で、`kv key list` にゴミキーが増えない
- [ ] GET /api/bow に一昨日以前の date を渡すと 400
- [ ] 自分の点が自分の画面でタップ対象になっていない（dots から自ID除外）

## 5. 非追跡・プライバシー監査

### ページ側
- [ ] 初回ロードの Network タブ: 同一オリジン + fonts.googleapis.com / fonts.gstatic.com **のみ**
- [ ] Application タブ: **Cookie 0件**。localStorage 以外のストレージ（IndexedDB等)未使用
- [ ] 分析タグ・エラートラッキング（GA / Sentry 等）のコードが存在しない

### Worker側（grep監査）
- [ ] `scheduled` ハンドラが存在しない（cronゼロ）
- [ ] KV `delete` 呼び出しが存在しない（消すのはTTLだけ）
- [ ] `CF-Connecting-IP` がレート制限Mapの一時判定以外（put / console.log / レスポンス）に現れない
- [ ] `console.log` が本番コードに残っていない
- [ ] km / min / distance / duration 等、記録データのフィールド名が存在しない
- [ ] 会釈を数えるコード（カウンタ・配列蓄積・履歴）が存在しない
- [ ] wrangler.toml に `[observability]` / `[triggers]` が存在しない。ダッシュボードでも Logs / Analytics Engine が無効のまま
